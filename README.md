# Snom-Remote-Control

Snom-Remote-Control ist eine Anwendung zur Steuerung von Snom Telefonen.

### Features
✔ Abheben

✔ Auflegen

✔ Wählen

✔ DND

✔ Transfer

✔ Halten

✔ Lautstäre steuern

✔ Headset/Lautsprecher/Hörer umschalten

### Unterstützte Modelle
✔ snom3xx

✔ snom7xx

✔ snom8xx
